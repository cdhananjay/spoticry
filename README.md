# Spoticry
A full stack next.js website which roasts people based on their spotify data (top artists and top tracks).

### Tech Stack
- Next.js
- Tailwind CSS
- Better-Auth
- Spotify API
- Gemini API

### Getting started
```
git clone https://github.com/cdhananjay/spoticry
cd spoticry
npm install
# setup .env file
npm run dev 
```

### Customization
Edit `@/lib/generatePrompts.ts` for custom prompts.
